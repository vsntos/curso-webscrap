Quando Nada Mais Funciona

Selenium é o último recurso

Use Selenium quando:

  • o conteúdo exige JS obrigatório;
  • há CAPTCHA ou verificação humana;
  • login é necessário;
  • a interação envolve cliques e formulários;
  • scroll infinito não pode ser interceptado.

O que é Selenium

Framework para controlar navegadores reais via código.

O navegador executa JS, carrega cookies, renderiza tudo.

Arquitetura do RSelenium

Componentes

R → RSelenium → WebDriver → Navegador (Chrome/Firefox)
  • R — nosso código;
  • RSelenium — pacote que comunica com o WebDriver;
  • WebDriver — protocolo de automação;
  • Navegador — Chrome, Firefox, ou headless.

Scroll Infinito

Padrão de coleta

coletar_com_scroll <- function(url, n_scrolls = 10) {

  remDr$navigate(url)
  Sys.sleep(3)

  todos_titulos <- character(0)

  for (i in seq_len(n_scrolls)) {

    html_atual <- remDr$getPageSource()[[1]]
    pagina <- read_html(html_atual)

    titulos <- pagina |>
      html_elements(".titulo") |>
      html_text2()

    novos <- setdiff(titulos, todos_titulos)
    todos_titulos <- unique(c(todos_titulos, titulos))

    cat(sprintf("Scroll %d: +%d títulos (%d total)\n",
                i, length(novos), length(todos_titulos)))

    # Scrollar
    remDr$executeScript(
      "window.scrollTo(0, document.body.scrollHeight);"
    )
    Sys.sleep(2)

    if (length(novos) == 0) {
      cat("Sem novos elementos — encerrando.\n")
      break
    }
  }

  todos_titulos
}

Navegação Entre Páginas

Clicar em “próxima página”

coletar_multiplas_paginas <- function(url, max_paginas = 20) {

  remDr$navigate(url)
  Sys.sleep(2)

  resultados <- list()

  for (p in seq_len(max_paginas)) {

    # Extrair dados da página atual
    html <- remDr$getPageSource()[[1]]
    dados <- extrair_dados(read_html(html))
    resultados[[p]] <- dados

    cat(sprintf("Página %d: %d itens\n", p, nrow(dados)))

    # Procurar botão de próxima página
    proxima <- tryCatch(
      remDr$findElement("css selector", "a[rel='next'], .proxima-pagina"),
      error = function(e) NULL
    )

    if (is.null(proxima)) {
      cat("Última página.\n")
      break
    }

    proxima$clickElement()
    Sys.sleep(2)
  }

  bind_rows(resultados)
}

Aguardar Elementos

Espera explícita

aguardar_elemento <- function(remDr, seletor, timeout = 10) {

  inicio <- Sys.time()

  repeat {
    elemento <- tryCatch(
      remDr$findElement("css selector", seletor),
      error = function(e) NULL
    )

    if (!is.null(elemento)) return(elemento)

    if (as.numeric(Sys.time() - inicio) > timeout) {
      stop(sprintf("Timeout: elemento '%s' não apareceu.", seletor))
    }

    Sys.sleep(0.5)
  }
}

scraper_selenium <- function(url) {

  driver <- rsDriver(browser = "chrome", verbose = FALSE)
  remDr  <- driver$client

  on.exit({
    remDr$close()
    driver$server$stop()
  })

  # ... scraping ...
}

Exemplo Completo: Portal Legislativo

Câmara dos Deputados — votações

coletar_votacoes_camara <- function(legislatura = 57) {

  driver <- rsDriver(browser = "chrome", verbose = FALSE, port = 4445L)
  remDr  <- driver$client
  on.exit({ remDr$close(); driver$server$stop() })

  url <- paste0(
    "https://www.camara.leg.br/votacoes#",
    "legislatura=", legislatura
  )

  remDr$navigate(url)
  Sys.sleep(4)

  resultados <- list()

  for (p in 1:5) {

    html <- remDr$getPageSource()[[1]]
    pagina <- read_html(html)

    votos <- pagina |>
      html_elements(".pauta-votacao") |>
      html_text2()

    if (length(votos) > 0) {
      resultados[[p]] <- tibble(votacao = votos, pagina = p)
    }

    # Próxima página
    btn_prox <- tryCatch(
      remDr$findElement("css selector", ".paginacao-proxima"),
      error = function(e) NULL
    )

    if (is.null(btn_prox)) break

    btn_prox$clickElement()
    Sys.sleep(2)
  }

  bind_rows(resultados)
}

Boas Práticas com Selenium

Respeitar o servidor

  • delays entre ações (Sys.sleep);
  • simular comportamento humano;
  • não raspar em paralelo agressivo;
  • verificar robots.txt.

Performance

  • preferir headless quando não precisa ver;
  • fechar sessão sempre que terminar;
  • não abrir múltiplos drivers desnecessariamente.

Exercício

Automatizar navegação com RSelenium

Escolha um portal legislativo ou governamental.

  1. Instale RSelenium e configure o Chrome;
  2. Navegue para a URL alvo;
  3. Extraia o HTML após renderização;
  4. Raspe dados de pelo menos 3 páginas clicando em “próxima”;
  5. Salve em CSV;
  6. Feche o driver com on.exit().
