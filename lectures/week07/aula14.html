Framing em Política Internacional

O que é framing?

A seleção e ênfase de aspectos de uma realidade percebida.

Como um mesmo evento é narrado de formas diferentes?

Exemplo empírico

library(quanteda)
library(tidyverse)

titulos <- c(
  # Portal A: foco em cooperação
  "Brasil e China ampliam parceria estratégica",
  "BRICS avança em integração financeira",
  "América do Sul fortalece soberania econômica",
  # Portal B: foco em tensão

  "Dependência do BRICS do yuan preocupa analistas",
  "Multipolaridade desafia ordem liberal"
)

portal <- c(rep("Portal_Sul", 3), rep("Portal_Norte", 3))

corp <- corpus(titulos)
docvars(corp, "portal") <- portal

Medindo framing com keyness

dfm_p <- corp |>
  tokens(remove_punct = TRUE) |>
  tokens_remove(stopwords("pt")) |>
  dfm() |>
  dfm_group(groups = portal)

textstat_keyness(dfm_p, target = "Portal_Sul")

noticias |>
  mutate(
    ukraine = str_detect(titulo, regex("ucrânia|ukraina", ignore_case = TRUE)),

GDELT Project

A maior base de eventos internacionais

GDELT monitora transmissões de notícias em 100+ idiomas.

Codifica:

  • evento — tipo de ação (cooperação, conflito, etc.);
  • atores — quem fez o quê a quem;
  • localização — onde ocorreu;
  • tom — positivo ou negativo.

API do GDELT

library(httr2)
library(jsonlite)

buscar_gdelt <- function(query, max_records = 25) {

  url <- paste0(
    "https://api.gdeltproject.org/api/v2/doc/doc",
    "?query=", URLencode(query),
    "&mode=artlist",
    "&maxrecords=", max_records,
    "&format=json"
  )

  resp <- tryCatch(
    request(url) |>
      req_user_agent("Curso RI / pesquisa academica") |>
      req_timeout(20) |>
      req_perform(),
    error = function(e) NULL
  )

  if (is.null(resp)) return(tibble())

  dados <- resp |> resp_body_string() |> fromJSON()

  if (is.null(dados$articles)) return(tibble())

  as_tibble(dados$articles)
}

Séries Temporais de Atenção

Atenção midiática como variável

# Monitorar cobertura por semana
monitorar_atencao <- function(tema, semanas = 8) {

  datas <- seq.Date(
    Sys.Date() - weeks(semanas),
    Sys.Date(),
    by = "week"
  )

  map_dfr(datas, function(d) {

    Sys.sleep(1)

    url <- paste0(
      "https://news.google.com/rss/search?q=",
      URLencode(tema),
      "&after=", format(d, "%Y-%m-%d"),
      "&hl=pt-BR&gl=BR"
    )

    rss <- tryCatch(xml2::read_xml(url), error = function(e) NULL)
    if (is.null(rss)) return(tibble())

    n <- length(xml2::xml_find_all(rss, "//item"))
    tibble(tema = tema, semana = d, n_noticias = n)
  })
}

Análise de Redes

Quem cita quem? Quem menciona quem?

library(quanteda)

titulos_rede <- c(
  "EUA e OTAN discutem apoio à Ucrânia",
  "China e Rússia fortalecem parceria estratégica",
  "Brasil e China assinam acordo no BRICS",
  "OTAN e EUA pressionam aliados europeus",
  "Rússia e Irã coordenam posição na ONU"
)

# Extrair co-menções de países
paises <- c("EUA", "OTAN", "China", "Rússia", "Brasil", "Irã", "Ucrânia")

co_mencoes <- map_dfr(titulos_rede, function(t) {
  presentes <- paises[str_detect(t, paises)]
  if (length(presentes) < 2) return(tibble())
  tibble(de = presentes[1], para = presentes[2], titulo = t)
})
