Web Scraping e Dados Digitais em RI

O Problema

Scraping produz listas

Dados brutos de scraping chegam como:

  • vetores de texto;
  • listas aninhadas;
  • texto sujo;
  • datas inconsistentes.

Precisamos transformar em datasets

Um dataset científico tem:

  • uma linha por observação;
  • uma coluna por variável;
  • tipos corretos;
  • sem duplicatas;
  • sem sujeira.

Da Coleta ao Dataset

O pipeline

scraping → lista → tibble → limpeza → normalização → análise

Limpeza de Texto

Problemas comuns

textos_brutos <- c(
  "  \n Diplomacia brasileira \t ",
  "ONU debate crise\n\n",
  "China supera EUA    em exportações"
)

str_squish() — o mais útil

library(stringr)

str_squish(textos_brutos)

Outras funções essenciais

x <- "  Líderes do G7 se reúnem em Roma  "

str_trim(x)              # remove bordas
str_to_lower(x)          # minúsculas
str_to_upper(x)          # maiúsculas
str_to_title(x)          # título
str_remove(x, "do G7 ")  # remove padrão

Expressões Regulares

Padrões para limpeza

titulos <- c(
  "Notícia: OTAN amplia presença",
  "Breaking: China e EUA negociam acordo",
  "Exclusivo: Itamaraty divulga nota"
)

# Remover prefixos padrão
str_remove(titulos, "^[A-Za-záéíóú]+: ")

Casos comuns em RI

# Remover numeração
str_remove("1. Acordo firmado em Genebra", "^\\d+\\.\\s*")

# Extrair ano de uma string
str_extract("Acordo de 2015 renovado", "\\d{4}")

# Detectar menção a países
str_detect(
  c("Brasil negocia com China", "EUA sanciona Irã"),
  "Brasil|China"
)

Normalização de Datas

Datas chegam em formatos variados

datas_brutas <- c(

  "2026-05-11",
  "May 10, 2026",
  "10/05/2026"

lubridate facilita o parsing

library(lubridate)

# Formato ISO já funciona direto
ymd("2026-05-11")

# Formatos mistos
parse_date_time(
  c("12 mai 2026", "May 10, 2026"),
  orders = c("d b Y", "B d, Y"),
  locale = c("pt_BR", "en_US")
)

Formatos mais comuns

Função Formato

Tipos de Dados

Converter após scraping

df_bruto <- tibble(
  titulo     = c("Notícia A", "Notícia B"),
  data_texto = c("2026-05-12", "2026-05-11"),
  paginas    = c("12", "34"),
  relevante  = c("sim", "não")
)

df_tipado <- df_bruto |>
  mutate(
    data      = ymd(data_texto),
    paginas   = as.integer(paginas),
    relevante = relevante == "sim"
  ) |>
  select(-data_texto)

df_tipado

Deduplicar

Notícias aparecem em múltiplas páginas

df_com_dup <- tibble(
  titulo = c("ONU debate", "ONU debate", "OTAN amplia"),
  link   = c("/a", "/a", "/b")
)

# Deduplica pelo link (mais confiável que título)
distinct(df_com_dup, link, .keep_all = TRUE)

Filtrar Ruído

Menus, rodapés e outros elementos

df_sujo <- tibble(
  titulo = c(
    "Internacional",
    "ONU aprova resolução histórica",
    "Leia mais",
    "China e EUA debatem tarifas",
    "Compartilhar"
  )
)

# Remover textos curtos ou genéricos
stopwords_ui <- c("Internacional", "Leia mais", "Compartilhar",
                  "Veja também", "Voltar", "Menu")

df_sujo |>
  filter(
    nchar(titulo) > 20,
    !titulo %in% stopwords_ui
  )

Enriquecer o Dataset

Adicionar colunas derivadas

df_noticias <- tibble(
  titulo = c(
    "ONU discute reforma do Conselho de Segurança",
    "BRICS avança em moeda comum",
    "COP30 define metas climáticas"
  ),
  data = ymd(c("2026-05-12", "2026-05-11", "2026-05-10"))
)

df_enriquecido <- df_noticias |>
  mutate(
    # Componentes de data
    ano      = year(data),
    mes      = month(data, label = TRUE),
    dia_sem  = wday(data, label = TRUE),

    # Tamanho do título
    n_chars  = nchar(titulo),
    n_words  = str_count(titulo, "\\S+"),

    # Menção a atores
    menciona_onu    = str_detect(titulo, regex("ONU|Nações Unidas", ignore_case = TRUE)),
    menciona_brics  = str_detect(titulo, regex("BRICS", ignore_case = TRUE))
  )

df_enriquecido

Pipeline Completo

Do scraping ao dataset limpo

library(rvest)
library(tidyverse)
library(lubridate)

# 1. Coletar
paginas <- paste0(
  "https://agenciabrasil.ebc.com.br/internacional?page=",
  0:4
)

df_bruto <- purrr::map_dfr(paginas, function(url) {
  Sys.sleep(1.5)
  pagina <- tryCatch(read_html(url), error = function(e) NULL)
  if (is.null(pagina)) return(tibble())
  tibble(
    titulo = pagina |> html_elements("h2") |> html_text2(),
    link   = pagina |> html_elements("h2 a") |> html_attr("href"),
    coletado = Sys.time()
  )
})

# 2. Limpar
df_limpo <- df_bruto |>
  mutate(titulo = str_squish(titulo)) |>
  filter(nchar(titulo) > 20) |>
  distinct(link, .keep_all = TRUE)

# 3. Salvar
write_csv(df_limpo, "data/processed/noticias_internacionais.csv")

BRICS avança… 2026-05-11 AB /noticias/… 4 FALSE

Exercício

Construir dataset com 100+ observações

  1. Escolha um portal;
  2. Raspe pelo menos 5 páginas;
  3. Limpe títulos com str_squish();
  4. Normalize datas com lubridate;
  5. Adicione colunas derivadas (ano, mês, n_words);
  6. Deduplicar por link;
  7. Filtre ruído (nchar > 20);
  8. Salve em CSV.
