O Pacote rvest

Criado pelo Hadley Wickham

rvest é o pacote padrão para scraping em R.

  • integrado ao tidyverse;
  • sintaxe simples e legível;
  • funciona com CSS selectors e XPath.
library(rvest)

Funções Principais

Leitura

Função Uso
read_html() lê uma página HTML

Seleção

Função Uso
html_elements() todos os elementos
html_element() apenas o primeiro

Extração

Função Uso
html_text2() texto do elemento
html_attr() atributo específico
html_table() tabela HTML

Fluxo Básico de Scraping

Três etapas

# 1. Ler a página
pagina <- read_html("https://site.com")

# 2. Selecionar elementos
elementos <- pagina |>
  html_elements(".classe-do-elemento")

# 3. Extrair conteúdo
textos <- elementos |>
  html_text2()

Texto vs Atributo

Quando usar cada um?

# Texto visível do elemento
html_elements(pagina, "h2") |>
  html_text2()

# Atributo href (endereço do link)
html_elements(pagina, "a") |>
  html_attr("href")

# Atributo src (endereço da imagem)
html_elements(pagina, "img") |>
  html_attr("src")

# Atributo class
html_elements(pagina, "div") |>
  html_attr("class")

Tratamento de Erros

Páginas que falham

raspar_pagina <- function(url) {

  tryCatch(
    {
      Sys.sleep(1)
      read_html(url)
    },
    error = function(e) {
      message("Falhou: ", url)
      return(NULL)
    }
  )

}

Filtrar resultados nulos

resultados <- purrr::map(urls, raspar_pagina)

# Remover falhas
resultados_validos <- purrr::compact(resultados)

Iterar com purrr

extrair_titulos <- function(url) {

  Sys.sleep(1.5)

  pagina <- tryCatch(
    read_html(url),
    error = function(e) return(NULL)
  )

  if (is.null(pagina)) return(tibble())

  tibble(
    titulo = pagina |>
      html_elements("h2") |>
      html_text2(),
    url_fonte = url
  )
}

df_todas <- purrr::map_dfr(paginas, extrair_titulos)

Exemplo Completo

Do zero ao dataset

library(rvest)
library(tidyverse)

# 1. URL alvo (verificar robots.txt antes)
url <- "https://agenciabrasil.ebc.com.br/internacional"

# 2. Ler página
pagina <- read_html(url)

# 3. Extrair elementos
titulos <- pagina |>
  html_elements("h2") |>
  html_text2() |>
  stringr::str_squish()

links <- pagina |>
  html_elements("h2 a") |>
  html_attr("href")

# 4. Construir dataset
df <- tibble(
  titulo     = titulos,
  link       = links,
  coletado   = Sys.time()
) |>
  filter(nchar(titulo) > 5)

# 5. Salvar
write_csv(df, "noticias_internacionais.csv")

df

Quando rvest Não Funciona

Páginas dinâmicas (JavaScript)

Alguns sites renderizam conteúdo com JavaScript.

read_html() captura apenas o HTML inicial — sem JS.

# rvest captura isso:
<div id="noticias"></div>

# Mas o conteúdo só aparece DEPOIS do JS rodar:
<div id="noticias">
  <article>...</article>
  <article>...</article>
</div>

Solução: Selenium

Para conteúdo dinâmico usamos RSelenium.
