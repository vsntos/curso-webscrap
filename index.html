Web Scraping e Dados Digitais em RI

Métodos computacionais para pesquisa em Relações Internacionais

Bem-vindo ao curso

Aprenda a coletar, organizar e analisar dados digitais para pesquisa em RI — de feeds RSS e APIs governamentais a scraping dinâmico, pipelines reprodutíveis e análise de texto com quanteda.

Ver Ementa Ver Cronograma

O que você vai aprender

Coleta de Dados

  • HTML, CSS e DevTools
  • Requisições HTTP com httr2
  • Scraping estático com rvest
  • Scraping dinâmico com RSelenium e chromote
  • APIs: Banco Mundial, GDELT, ONU Comtrade

Infraestrutura

  • Organização de projetos reprodutíveis
  • Ambientes com renv
  • Pipelines declarativas com targets
  • Logging, retry e tratamento de erros
  • Git para projetos de dados

Análise

  • NLP com quanteda
  • Análise de framing e agenda-setting
  • Redes de co-menção com igraph
  • Integração dados textuais + econômicos
  • Relatórios automáticos com Quarto

Estrutura do curso

O curso tem 15 aulas organizadas em 8 semanas temáticas:

Semana Tema Aulas
1 Fundamentos — dados digitais em RI, ética 1–2
2 Web Foundations — HTML, CSS, HTTP, APIs 3–4
3–4 Scraping Estático — rvest, limpeza, loops, robustez 5–8
5 Scraping Dinâmico — JavaScript, Selenium 9–10
6 Infraestrutura — organização, targets 11–12
7 Texto e NLP — quanteda, framing, redes 13–14
8 Projeto Final — apresentações 15

Ferramentas

install.packages(c(
  "tidyverse", "rvest", "httr2", "jsonlite", "xml2",
  "RSelenium", "chromote", "targets", "tarchetypes",
  "quanteda", "quanteda.textstats", "quanteda.textplots",
  "renv", "fs", "usethis", "lubridate", "purrr"
))
